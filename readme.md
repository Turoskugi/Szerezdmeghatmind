# 🎮 Fortnite Sprite Tracker

Egy interaktív webes alkalmazás, ahol te és a barátaid közösen nyomon követhetitek a Fortnite spriteöket!

## ✨ Funkciók

- ✅ **Felhasználói fiókok** - Regisztráció és bejelentkezés
- ✅ **Sprite Katalógus** - Összes Fortnite sprite (skinok, back blings, csákányok, stb.)
- ✅ **Gyűjtőlista** - Jelöld meg melyik sprite van meg neked
- ✅ **Valós idejű szinkronizáció** - Automatikusan menti a jelöléseket
- ✅ **Szűrés és kategorizálás** - Keress kategória alapján
- ✅ **Statisztika** - Lásd a gyűjtés végzettségét
- ✅ **LocalStorage Cache** - Offline támogatás

## 🚀 Telepítés

### Előfeltételek
- Node.js (14+)
- MongoDB (local vagy Atlas)
